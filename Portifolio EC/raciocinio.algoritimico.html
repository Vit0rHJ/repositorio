Projeto Jokenpõ

Desenvolvido em Python puro, este projeto oferece três modalidades de jogo: Humano vs Humano, Humano vs Computador e Computador vs Computador, com sistema de placar e validação de entradas. Utiliza random.choice() para jogadas aleatórias da IA e estruturas condicionais para determinar o vencedor conforme as regras clássicas (Pedra > Tesoura > Papel > Pedra). O código apresenta tratamento de erros para inputs inválidos, menu interativo no terminal e opção de trocar de modalidade entre partidas. Ideal para demonstrar lógica de programação básica e controle de fluxo, com potencial para expansão com novas jogadas (como Lagarto-Spock) ou interface gráfica. Esse projeto foi meu primeiro contato mais aprofundado com qualquer linguagem de programação, foi muito divertido e aprendi muito da base de python com ele.

Projeto Maquina de vendas

Desenvolvido em Python, este projeto simula uma máquina de vendas inteligente com dois modos de operação: cliente (para compras) e administrador (para gestão do sistema). Oferece controle de estoque de produtos (como bebidas) e gerenciamento de troco (notas e moedas), com funções para cadastro, edição e remoção de itens. O sistema calcula troco automaticamente, priorizando as cédulas disponíveis, e valida entradas para evitar erros. Ideal para demonstrar lógica de programação avançada, manipulação de matrizes e gestão de dados, com potencial para integração com banco de dados ou interface gráfica. Esse foi um projeto relativamente desafiador, porem de todos os que realizei foi oque me ensinou mais coisa com certeza.
